<h2> <span class="mw-headline" id="Misc._Comments">Misc. Comments</span></h2>
<p>This file is probably not from W.F. Bach because it also can be found in a collection of pieces from J.S. Bach (BWV Anh. 133) but Martin Falck added this file as F 22 to the work list of W.F. Bach
</p>
<ul><li> Grove Music lists the composition date for this as "?c1750". 
</li><li> Grove Music lists Wilhelm Friedemann Bach's keyboard works as "Keyboard, organ". No mention is made of piano, so it is unlikely that this was originally intended for piano. 
</li><li> Also known as Fantasia BWV Anh. 133 from Johann Sebastian Bach
</li></ul>
